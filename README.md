# 🛍️ Customer Shopping Behavior Analysis

## 📌 Description

This project focuses on analyzing customer shopping behavior using **MySQL, Python, and Power BI** to generate actionable business insights.

The dataset consists of customer demographics, purchase details, product categories, discounts, and subscription information. Using MySQL, structured queries were written to extract insights related to revenue, customer segmentation, and product performance.

Key analyses include:

* Revenue comparison across different customer segments (gender, age group)
* Impact of discounts on purchase behavior
* Identification of top-performing products based on ratings and sales
* Customer segmentation into New, Returning, and Loyal customers
* Comparison between subscribed and non-subscribed users

Additionally, exploratory data analysis (EDA) was performed using Python to better understand patterns and trends in the dataset.

An interactive **Power BI dashboard** was created to visualize:

* Revenue distribution
* Customer segmentation
* Product performance
* Purchasing trends

---

## 🎯 Objective

To analyze customer purchase data using MySQL and visualization tools to extract insights that support data-driven business decisions.
